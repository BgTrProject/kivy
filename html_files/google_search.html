How to use Google Search


Google üzerinden arama yapmamızı sağlar.
Arama yapmak için ilk önce arama yapacağımız konuyu keyword kısmına girmemiz gerekir.
Link kısmı hangi link üzerinden arama yapmak istiyorsak o link girilir(http://www.nytimes.com gibi).Aramaya başlanacak tarih ise start date kısmıdır burayı doldurduktan sonra hangi tarihe kadar arama yapılmak isteniyorsa end date girilir.Sonra ise arama sonuçlarını hangi dosya adıyla kaydedilmek isteniyorsa filename kısmı doldurulup arama butonuna basılır.
